Parameter Data type Description Example
grant_type string Used to assess the method to grant authorization. authorization_code
code string Used as the authorization_code return in the callback from the authorize endpoint. labwuigwbweogbnoiweg
client_id string Used to identify a client. 8BDB0EC9-32C0-4324-B836-B41538471A8E
client_secret string Used to authenticate a client. sdbvisdbvsiudbvisvbsd
redirect_uri string Used to match the redirect_uri in the authorize request. https://localhost:5001/callback
scope space delimited string Used to assess the usage of the access token. Must include openid. openid identityprovider:userinfo weather:write
code_verifier string Used to verify the code_challenge was generated from the code_verifier. isubvisvubiusebvsiev
refresh_token string Used to get a new access_token in the refresh_token grant flow. ey
