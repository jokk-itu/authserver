Blazor BFF

Introduction

A client built using Blazor WebAssembly. It contains of a Client compiled into WebAssembly and is the Frontend in BFF. An API responsible for Authentication and Authorization, and is the Backend in BFF. The API also functions as a Reverse Proxy for the Client.
